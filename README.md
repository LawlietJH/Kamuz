### Versión Actual: v1.1.0 (Con Interfaz Gráfica)
- - -
### Python: 3 
### OS: Windows
- - -
### Dependencias: PyWin32 y PyQT5.

  * __Se Debe Instalar Desde Una Terminal (CMD) Con el Comando: _python -m pip install -r requirements.txt___
